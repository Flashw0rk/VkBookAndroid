   VkBookAndroid v1.0 - ИТОГОВАЯ СВОДКА ПРОГРАММЫ

📱 ЧТО ЭТО:

Мобильное приложение для работников реакторной установки ВК-50.
Позволяет смотреть расписание смен, график проверок оборудования,
данные по арматуре и сигналам БЩУ, работать с документацией.

═══════════════════════════════════════════════════════════

⚙️ ОСНОВНЫЕ ФУНКЦИИ:

1. АРМАТУРА
   → База данных арматуры с поиском
   → Просмотр схем и характеристик
   
2. СИГНАЛЫ БЩУ
   → Справочник сигналов блочного щита управления
   → Быстрый поиск по номеру/описанию

3. ДАННЫЕ
   → Просмотр больших Excel таблиц
   → Кэширование для быстрой работы

4. РАСПИСАНИЕ
   → Календарь смен на годы вперед (2025-2030)
   → График смен синхронизирован с календарем
   → Выбор месяца и даты

5. ГРАФИК ПРОВЕРОК
   → Расписание проверок оборудования по часам
   → Личные задачи и служебные
   → Правила напоминаний (когда делать проверки)
   → Подсветка текущего часа

6. НАСТРОЙКИ
   → 6 профессиональных тем оформления
   → Подключение к серверу
   → Автосинхронизация
   → Выбор видимых вкладок

═══════════════════════════════════════════════════════════

🔧 ТЕХНОЛОГИИ:

Язык: Kotlin
UI: Material 3, ViewPager2, RecyclerView
Сеть: Retrofit, OkHttp
Excel: Apache POI
Фон: WorkManager
Данные: SharedPreferences + File cache
Тесты: JUnit, Espresso, Robolectric
