📱 УСТАНОВКА НА ТЕЛЕФОН/ПЛАНШЕТ:

СПОСОБ 1: Через компьютер (ADB)
────────────────────────────────
1. Подключить устройство к компьютеру (USB)
2. Включить "Отладка по USB" в настройках разработчика
3. Открыть командную строку в этой папке
4. Выполнить: adb install VkBookAndroid-v1.0-release.apk

СПОСОБ 2: Прямая установка
───────────────────────────
1. Скопировать APK на устройство (через USB/облако/почту)
2. На устройстве открыть файл-менеджер
3. Найти файл VkBookAndroid-v1.0-release.apk
4. Нажать на файл
5. Разрешить установку из неизвестных источников (если спросит)
6. Нажать "Установить"

СПОСОБ 3: Через Android Studio
───────────────────────────────
1. Открыть проект VkBookAndroid
2. Build Variants → release
3. Run → Run 'app'
